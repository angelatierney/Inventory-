# Inventory Manager-
A lightweight Java application designed to track and manage store inventory. This tool allows for efficient monitoring of stock levels, product details, and inventory updates.

Key Features
Add/Remove Items: Easily update the inventory list.

Stock Tracking: Monitor current quantities of products.

Java Core: Built using object-oriented principles for simplicity and speed.

Quick Start
Ensure you have Java installed.

Compile and run Main.java.

Follow the console prompts to manage your store's data.
